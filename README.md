# OSSMA

Offline-first professional assessment app for sport and health professionals.

## Status

Initial foundation.

## Tech Stack

- SvelteKit
- TypeScript
- pnpm
- Vitest
- ESLint
- Prettier
- adapter-static

## Development

```bash
pnpm install
pnpm run dev
pnpm run format
pnpm run lint
pnpm run test
pnpm run check
```
