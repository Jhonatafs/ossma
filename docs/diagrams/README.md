# Diagramas de Fluxo de Uso

## Arquivos

- `01-user-workflow.mmd`: fluxo geral da aplicação.
- `02-onboarding-flow.mmd`: primeiro acesso, idioma, tema e criação de perfil.
- `03-home-and-navigation-flow.mmd`: página inicial, dashboard e navegação universal.
- `04-client-management-flow.mmd`: gestão de clientes, anamnese e histórico.
- `05-evaluation-workflow.mmd`: fluxo completo de avaliação.
- `06-report-workflow.mmd`: pré-visualização, relatório, carimbo e exportação.
